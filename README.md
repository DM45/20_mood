# Site for the Mood

Small site with funny quote

# How to install

Copy this repository and host it

# How to add new quote

Add new quote to var quotes in index.html
```bash
{
 "phrase": "Кукурузные хлопья - теперь без асбеста!",
 "signature": "©Неизвестный маркетолог"
},
```
